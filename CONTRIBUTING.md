# Contributing to AI Ready PH Learning Hub

Thanks for your interest in making this better. This project is built for Filipino VAs, freelancers, and business owners — if you've used it and found something that could be improved, you're in the right place.

## Ways to Contribute

### 🐛 Report a Bug
- Open an issue with a clear title and steps to reproduce.
- Include what you expected and what actually happened.
- Screenshots or screen recordings help tremendously.

### 💡 Suggest a Feature or Lab
- Open an issue describing what you'd like to see and why it matters to learners.
- If it's a new lab: describe the scenario, what the learner does, and what competency it builds.

### 📝 Improve Curriculum or Documentation
- Typos, unclear lessons, outdated content — all welcome.
- PRs for README, lesson text, or lab instructions are reviewed promptly.

### 🧪 Submit a Lab Scenario
- Labs use synthetic data packs with a scenario brief, source materials, and success criteria.
- If you've designed a realistic work scenario that would make a good practice lab, share it.

## Pull Request Guidelines

1. Keep changes focused. One PR = one concern.
2. Write a descriptive title and explain *what* and *why*.
3. If the PR touches curriculum or learner experience, explain the pedagogical reason.
4. No AI-generated bulk PRs without human review — we'll spot it.
5. Be kind and constructive in discussions.

## Code of Conduct

Be respectful. This is a learning product — everyone started somewhere.

## Questions?

Open a discussion or reach out via the repository issues.
